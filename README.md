# python-nettle

Python bindings for the nettle low-level cryptographic library

## Getting Started


### Prerequisites

You'll need at least version 3 of nettle installed.

### Installing

```
cd src
python generator.py
cd ..
python setup.py install
```

Support for python 2 was dropped in 2025.1
